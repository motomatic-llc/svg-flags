Variants

Circle

512×512 circular flags. Geometry from circle-flags, <mask> → <clipPath>, official colors. Flags with white backgrounds include a <circle> border stroke (#cdcfd3).

View all →

Rect

512×512 square flags. Same circle-flags geometry without the circular clip. Flags with white backgrounds include a <rect> border stroke (#cdcfd3).

View all →

Full-size simplified

True aspect ratio, simplified geometry for complicated flags, official colors. No border strokes.

View all →

Full-size

True aspect ratio with accurate detail. Based on Wikimedia Commons SVGs. No border strokes.
